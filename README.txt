3. Cómo ejecutar el proyecto
----------------------------

### Requisitos previos

- **Node.js 20 LTS** (o versión indicada en `package.json`)
- **npm** (incluido con Node)

### Pasos

# 1. Instalar dependencias
/*Desde dentro de la carpeta del proyecto*/
npm install

# 2. Levantar el servidor de desarrollo
npm run dev
